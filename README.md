# 🚀 Auto Script Install X-ray & Zivpn + SSH WS

**EDITION: PLATINUM CLEAN V.6.0 (NO CLIENT BOT)**

Script instalasi otomatis (*Auto Installer*) untuk men-setup server VPN di VPS dengan mudah, cepat, dan tanpa ribet. Dirancang khusus untuk performa yang stabil, ringan, dan siap mendukung berbagai kebutuhan tunneling Anda.

## ✨ Fitur Utama

* **Multi-Protokol:** Mendukung penuh instalasi untuk protokol **SSH, VLESS, VMESS, dan Trojan**.
* **Integrasi Penuh:** Mendukung penggunaan panel manajemen langganan seperti **Marzban** serta terintegrasi dengan X-ray & Zivpn.
* **Clean Edition:** Versi ini telah dibersihkan dari Telegram Client Bot agar server lebih ringan dan fokus pada performa tunneling.
* **Auto-Restart Fix:** Perbaikan bug otomatis untuk menjaga stabilitas layanan VPS.
* **Anti-Interactive Install:** Proses instalasi berjalan otomatis tanpa gangguan pop-up dialog (*noninteractive*).
* **UI Menarik:** Dilengkapi dengan animasi *loading scanner* yang rapi saat proses instalasi berjalan.
